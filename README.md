# linky

## Backend

Provided by [@velushree](https://github.com/velushree)/[linky](https://github.com/velushree/linktree)

- FastAPI
- Mongodb

### Project setup

```
git clone https://github.com/velushree/linktree
```

```
pip install -r requirements.txt
```

### Run

```
uvicorn main:app --reload
```

## Frontend

- Vue js
- Materialize css

### Project setup
```
npm install
```

### Run
```
npm run serve
```
